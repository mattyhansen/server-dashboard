# Server Dashboard



### Install
if no pip: command not found, install:
```
sudo apt-get install python-dev
sudo apt-get install python-pip
sudo pip install psutil
sudo pip install psutil --upgrade
```

### Usage
```
sudo python monitor_json.py
```
